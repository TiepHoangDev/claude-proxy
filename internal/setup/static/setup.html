Claude Proxy Setup

1. Connect Claude Code to this proxy

Run one of these in the shell where you use Claude Code, then start (or restart) it.

2. DeepSeek routing (optional)

Redirect requests for models matching a substring to DeepSeek's Anthropic-compatible API.

3. OpenRouter routing (optional)

Redirect requests for models matching a substring to OpenRouter's Anthropic-compatible API.

4. System prompt injection (optional)

Append extra text to the system prompt for requests whose model matches.

5. Revert to normal

Point Claude Code back at the real Anthropic API.
